# RAPTOR SHOPPING SHORTS GEM - OPERATION GUIDE (V2.1)

## 1. 사용자 입력 안내

### 필수 입력
1. 상품명
2. 제품 이미지
3. 영상 길이

### 선택 입력
4. 상품 링크
5. 상세페이지 텍스트
6. 사용자 후기
7. 숏폼 주제 또는 컨셉
8. 말하기 속도
9. 타겟 사용자
10. 원하는 분위기 또는 스타일

---

## 2. 제품 이미지가 필수인 이유

제품 이미지는 다음에 직접 사용된다.

- Product_Ref 추출
- 장면별 제품 외형 일관성 유지
- 이미지 생성 정확도 향상
- 썸네일 대표 이미지 품질 향상
- 후속 영상 연결 품질 안정화

제품 이미지 없이는 품질이 떨어질 수 있으므로 반드시 요구한다.

---

## 3. 결과로 나오는 것

입력 후 다음 결과가 생성된다.

1. 상품 분석 요약
2. 구매 트리거
3. 선택된 Pattern
4. 숏폼 스크립트
5. Scene 구성
6. 장면별 이미지 프롬프트
7. Scene별 실제 생성 이미지
8. 업로드 패키지
   - 제목 5개
   - 설명문
   - 해시태그
   - 키워드
   - CTA 문장
   - 썸네일 문구 3~5개
   - 썸네일 대표 이미지

---

## 4. 역할 분담

### 기획자
- 시스템 구조 설계
- Main과 PLM 구성
- 생성 흐름 최적화
- 리뷰 요청

### 리뷰어
- 지침대로 실행 흐름 검토
- 입력값과 출력값 연결 검증
- Scene별 이미지 실제 생성 확인
- 썸네일 생성 결과 확인
- 문제 없으면 사용 가능 선언

---

## 5. 리뷰 절차

STEP 1: 필수 입력값 확인
- 상품명 있는가
- 제품 이미지 있는가
- 영상 길이 있는가

STEP 2: 상품 분석 확인
- Pain Point 추출되는가
- 핵심 장점 1개가 선명한가
- Product_Ref가 추출되는가

STEP 3: 전략 선택 확인
- 구매 트리거 선택되는가
- 적절한 Pattern 선택되는가
- Hook / WOW / CTA가 모두 선택되는가

STEP 4: 스크립트 확인
- 첫 문장이 강한가
- WOW가 후킹 직후 들어가는가
- CTA가 자연스럽게 이어지는가
- 영상 길이에 맞는 분량인가

STEP 5: Scene 확인
- Scene 수가 길이에 맞는가
- Scene마다 역할이 명확한가
- Scene 간 연결이 자연스러운가
- 제품 노출이 초반부터 이뤄지는가

STEP 6: 이미지 확인
- 각 Scene 이미지가 실제 생성되었는가
- 이미지 수가 Scene 수와 일치하는가
- Scene 목적과 이미지가 일치하는가
- 제품 외형 일관성이 유지되는가

STEP 7: 업로드 패키지 확인
- 제목 5개 생성되는가
- 설명문이 바로 사용 가능한가
- 해시태그 / 키워드가 자연스러운가
- 썸네일 문구가 강한가
- 썸네일 대표 이미지가 적절한가

---

## 6. 합격 기준

다음 항목이 모두 충족되면 사용 가능하다.

- Hook 강도 이상 없음
- WOW 구조 이상 없음
- CTA 구조 이상 없음
- Scene 구조 이상 없음
- 실제 이미지 생성 이상 없음
- 썸네일 이상 없음
- 길이 대응 이상 없음
- 업로드 패키지 이상 없음

---

## 7. 불합격 기준

다음 중 하나라도 있으면 수정 후 재검토한다.

- Scene 수와 이미지 수 불일치
- 이미지가 Scene 목적과 맞지 않음
- 제품 외형 일관성 붕괴
- 훅 약함
- WOW 없음
- CTA 어색함
- 썸네일 문구 약함
- 썸네일 이미지 대표성 부족

---
