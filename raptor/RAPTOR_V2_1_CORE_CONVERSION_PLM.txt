[RAPTOR V2.1 CORE CONVERSION PLM]

1. HOOK 사용 원칙
- 첫 문장은 반드시 강해야 한다
- 설명형보다 충격형, 금지형, 손해형, 비교형, 감정형을 우선한다
- 우선순위: Extreme Hook > Basic Hook

2. EXTREME HOOK LIBRARY
- 이거 쓰지 마세요
- 이거 아직도 이렇게 쓰세요
- 이 방식 계속 쓰면 손해입니다
- 이거 모르고 계속 쓰면 낭비입니다
- 편한 줄 알았죠 그게 더 비효율입니다
- 다들 편하다고 하는데 핵심은 따로 있습니다
- 이거 안 쓰면 계속 시간 버립니다
- 이거 모르고 계속 손해 보고 있었습니다
- 이거 전이랑 완전히 다릅니다
- 이거 쓰면 전으로 못 돌아갑니다
- 이거 왜 이제 알았죠
- 이거 보고 생각 바뀌었습니다
- 이거 때문에 진짜 짜증났습니다
- 이거 때문에 계속 스트레스였습니다
- 이거 사지 마세요 근데 이건 꼭 보세요
- 이거 별거 아닌 줄 알았는데 차이가 큽니다

3. BASIC HOOK LIBRARY
- 이거 때문에 계속 불편했습니다
- 이거 때문에 계속 고생했습니다
- 이거 안 쓰면 계속 손해입니다
- 이거 쓰기 전이랑 완전히 다릅니다
- 왜 이제 알았을까요
- 이거 하나로 일이 빨라집니다
- 이게 왜 인기인지 알겠습니다

4. WOW 사용 원칙
- 후킹 직후 1~2문장 안에 배치
- 단순 장점 나열 금지
- 새로운 발견, 의외성, 전후차이, 시간절약을 우선한다

5. WOW LIBRARY
- 이 기능은 거의 모릅니다
- 핵심은 이 기능입니다
- 다들 이 포인트를 놓칩니다
- 이렇게 쓰는 게 진짜 포인트입니다
- 생각보다 이 상황에서 더 빛납니다
- 이거 전이랑 완전히 다릅니다
- 사용 전후 차이가 바로 납니다
- 이거 하나로 시간 엄청 줄었습니다
- 찾는 시간 줄어드는 게 제일 큽니다
- 기존 방식은 그냥 낭비였습니다
- 문제는 제품이 아니라 방식이었습니다
- 한 번 쓰면 바로 체감됩니다

6. CTA 사용 원칙
- 정보 → 설득 → 행동 구조
- 무조건 강요하지 않는다
- 상황에 맞는 CTA 유형을 고른다

7. BASIC CTA LIBRARY
- 설명에 링크 있습니다
- 설명 참고하세요
- 설명에서 확인하세요
- 댓글에 링크 있습니다
- 댓글에서 확인하세요
- 고정 댓글에 링크 있습니다
- 관심 있으면 한번 확인해 보세요
- 필요한 분들은 참고하세요

8. CTA V2 LIBRARY
- 지금 가격 한번 확인해 보세요
- 가격은 설명에서 바로 확인 가능합니다
- 다른 제품이랑 비교해 보세요
- 이거랑 비교하면 차이가 바로 보입니다
- 필요한 분들은 지금 확인해 보세요
- 이런 제품 찾고 있었다면 한번 보세요
- 제품 정보는 설명에서 확인 가능합니다
- 댓글에 정보 남겨두겠습니다
- 궁금하면 설명에서 확인해 보세요
