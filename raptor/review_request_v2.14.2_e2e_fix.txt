[현재 KIE 키 저장 경로 - 수술 후]
경로 A (BYOKSettingsForm - Global Settings):
  입력 → api.post('/auth/set-key', kieKey) → 백엔드 쿠키 저장
        → setStoreKieKey(kieKey.trim()) → Zustand store.kieKey 저장  ← 이번에 추가
        → api-client.ts: headers['X-BYOK-KIE'] = store.kieKey → X-BYOK-KIE 헤더 전송 ✅

경로 B (AuthDashboard - 대시보드 설정 탭):
  입력 → store.setKieKey(kieKey) → Zustand store.kieKey 저장 (이전부터 정상)
        → api-client.ts: headers['X-BYOK-KIE'] = store.kieKey → X-BYOK-KIE 헤더 전송 ✅
