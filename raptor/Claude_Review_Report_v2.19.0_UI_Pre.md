2. **[용어 전면 교체]**:
   - Step 4 컴포넌트 전체에서 '렌더링'이라는 오용된 단어가 '비디오 생성'으로 정확히 일괄 치환되었습니다. '렌더링' 용어의 순수성이 복구되었습니다.

5. **[MP4 업로드 JS 관대화]**:
   - 엄격한 `!== 'video/mp4'` 유효성 검사 블록 두 군데가 모두 `!file.type.startsWith('video/')`로 관대하게 교체되어 Mac이나 iOS에서의 이종 MIME 타입 거부 현상이 소멸되었습니다.
