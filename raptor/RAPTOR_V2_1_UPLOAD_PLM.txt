3. 해시태그 생성 원칙
- 상품 카테고리
- 핵심 문제
- 사용 상황
- 플랫폼 키워드
를 반영한다

예시
#생활용품
#꿀템
#추천템
#생활아이템
#쿠팡추천

4. 키워드 생성 원칙
- 상품명
- 제품 유형
- 문제 상황
- 핵심 효용
- 타겟 사용자
를 기준으로 만든다

5. CTA 마무리 예시
- 설명에 링크 있습니다
- 설명 참고하세요
- 설명에서 확인하세요
- 댓글에 링크 있습니다
- 지금 가격 한번 확인해 보세요
- 다른 제품이랑 비교해 보세요

6. 썸네일 생성 규칙
- 썸네일은 업로드 패키지의 일부다
- 대표 Scene 기반으로 생성한다
- 대표 이미지는 한 장만 우선 생성한다
- 문구는 3~5개 생성한다

7. 썸네일 대표 이미지 선택 기준
- Hook가 가장 강한 장면
- 제품이 잘 보이는 장면
- 전후 차이가 강한 장면
- 놀람 또는 변화가 잘 드러나는 장면

8. 최종 업로드 패키지 구성
- 제목 5개
- 설명문
- 해시태그
- 키워드
- CTA 문장
- 썸네일 문구 3~5개
- 썸네일 대표 이미지 1장
