# SSPS (지능형 쇼핑 숏폼 상품 선정 시스템)

> **🔖 공식 버전: `v2.0`** — 이 파일이 버전의 유일한 기준점(Single Source of Truth)입니다.
> AI 보고서/기획서 작성 시 반드시 이 파일의 버전을 먼저 확인 후 기재할 것.

사용자가 분야를 입력하면 데이터 소스의 특성을 파악해 동적 가중치를 부여, 최적의 숏폼 상품군을 추천하고 RAPTOR GEM 통합 JSON을 출력하는 지능형 파이프라인.

## 🚀 빠른 시작 (Windows)

1. `run.bat` 파일을 더블 클릭하여 실행합니다.
2. 서버가 실행되면 (기본 포트 8000), 브라우저가 자동으로 대시보드를 엽니다.
3. 브라우저에서 `http://localhost:8000` 접속

## 🧠 에이전트 역할 분담
본 시스템은 4명의 AI 에이전트가 협업하여 구축되었습니다:
- **기획자 (Planner)**: 카테고리별 동적 가중치 공식 수립
- **개발자 (Developer)**: 병렬 커넥터, 회로 차단기, 점수 산출 엔진 개발
- **디자이너 (Designer)**: 글래스모피즘 기반 대시보드 및 JSON 스키마 시각화
- **리뷰어 (Reviewer)**: 통합 런타임 테스트 및 Circuit Breaker 폴백 룰렛 확인

## 🔑 .env 설정
네이버의 장기 트렌드 데이터를 활용하려면 `backend/config.py` 또는 직상단 `.env`에 검색 API 키가 필요합니다.
```env
USE_MOCK_DATA=false
NAVER_CLIENT_ID=your_id
NAVER_CLIENT_SECRET=your_secret
```
키가 없으면 `USE_MOCK_DATA=true` 로 전환되어 시뮬레이션 모드로 동작합니다.
